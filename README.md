# Tic Tac Toe game.

A **Tic Tac Toe** game made in **python**, using **panda3d** for the graphical interface.

The game supports only **player against player** mode.

## Features
- Graphical interface developed with Panda3D.
- Game logic implemented in Python.

## Requirements
- Python 3.8+.
- Panda3D.

## Install Panda3D
```bash
pip install panda3d
```

>The images must be in the same folder as TikTakToe.py.

## Clone, Build and Run
To use this project locally:
```bash
git clone https://github.com/Penpinch/Tic_Tac_Toe.git
python tikTakToe.py
```
